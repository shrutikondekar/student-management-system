# 🎓 Student Management System

A full-stack **Student Management System** built using **Spring Boot**, **React**, **PostgreSQL**, and **JWT Authentication**.

The application provides secure role-based authentication and allows administrators to manage student records efficiently through a modern React interface.

## ✨ Features

* 🔐 JWT Authentication
* 👥 Role-Based Authorization (ADMIN / USER)
* ➕ Add Student
* ✏️ Update Student
* ❌ Delete Student
* 🔍 Search Student
* 📄 Pagination
* ✅ Form Validation
* 🚨 Global Exception Handling
* 🔔 Toast Notifications
* 🛡️ Spring Security
